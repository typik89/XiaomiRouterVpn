# Xiaomi Router VPN Switch (неофициальное приложение)

Это **неофициальное** приложение для управления VPN на роутере Xiaomi.

Приложение позволяет **включать и выключать уже настроенный VPN** на роутере, если телефон подключен к **той же локальной сети**, что и роутер.

Проверено на роутере:
- **Xiaomi Mi Router 4A White (DVB4230GL) RTL 20, (525536)**

## Первый запуск

При первом запуске необходимо ввести настройки подключения, включая:
- адрес роутера;
- логин;
- пароль.

После сохранения эти параметры будут использоваться по умолчанию в следующих запусках приложения.
